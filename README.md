■ サービス概要

GoodsShop&Trade は、アイドルグッズの購入とトレード募集を一つのプラットフォームで実現するWebアプリケーションです。

ECサイトで新品グッズを購入できるだけでなく、生写真やランダムグッズの交換相手を探せる掲示板機能を統合しています。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

■ 開発背景

アイドルファンがグッズを購入する際、以下の課題がありました：

・EC機能とトレード機能が分散：購入サイトと交換募集サイトを別々に使う必要がある
・トレード相手を探す手間：SNSで個別に探すのが面倒
・安全性への不安：個人間取引の信頼性が低い

これらの課題を解決するため、ECサイトと掲示板を統合したプラットフォームを開発しました。

■ 主な機能

【ECサイト機能】
・商品一覧・詳細表示
・カート機能（非同期処理、リロード耐性）
・商品検索（AND/OR/NOT検索）
・ソート機能（価格順/人気順/売上順）
・お気に入り・いいね機能
・注文履歴管理

【トレード掲示板機能】
・トレード募集投稿
・画像アップロード（サムネイル自動生成）
・コメント機能
・カテゴリ別・月別アーカイブ
・SNS共有

【会員管理】
・ログイン/ログアウト（reCAPTCHA v3対応）
・会員登録/編集/退会
・パスワード再設定
・ソーシャルログイン
・マイページ（購入履歴、いいね管理）

【管理者機能】
・商品管理（CRUD、CSV対応）
・会員管理（検索、ソート、ページネーション）
・注文管理
・Basic認証
